# AI Sprint Health Tracker

A polished, frontend-only web demo designed to look like an intelligent engineering leadership dashboard. The app simulates pulling data from Jira and Git to present AI-generated insights for a single sprint.

- **Burnout Prediction**: Trend chart showing burnout risk over time.
- **Story Risk Analysis**: Detailed table highlighting at-risk stories with AI reasoning.
- **Workload Visualization**: Stacked bar chart showing developer activity balance.
- **Actionable Recommendations**: AI-generated suggestions to improve sprint health.
